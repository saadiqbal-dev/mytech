Google Analytics 4
UTM Link Builder

This tool allows you to easily add campaign parameters to URLs so you can measure Custom Campaigns in Google Analytics.

Fill out the required fields (marked with *) in the form below, and once complete the full campaign URL will be generated for you.

Note: the generated URL is automatically updated as you make changes.

The full website URL (e.g. https://www.example.com/landing-page-1)
Marketing Medium (e.g banner, email, social, affiliate, cpc etc)
The referring source (e.g. facebook, linkedin, instagram, newsletter, google etc )
Product, promo code, or slogan (e.g. spring_sale)
Use to differentiate ads
Identify the paid keywords
The ads campaign id

Share the generated campaign URL

Use this URL in any promotional channels you want to be associated with this custom campaign.

Need some help?

More information and examples for each parameter
The following table gives a detailed explanation and example of each of the campaign parameters:

Parameter Required Example Description
Campaign Medium
utm_medium 		Yes social Use utm_medium to identify a medium such as email or cost-per-click.
Campaign Source
utm_source 		Yes facebook Use utm_source to identify a search engine, newsletter name, or other source.
Campaign Name
utm_campaign 		No flash_sale Used for keyword analysis. Use utm_campaign to identify a specific product promotion or strategic campaign.
Campaign Term
utm_term 		No acme+widget Used for paid search. Useutm_term to note the keywords for this ad.
Campaign Content
utm_content 		No logolink Used for A/B testing and content-targeted ads. Use utm_content to differentiate ads or links that point to the same URL.
Campaign ID
utm_id 		No xyc.321 Used to identify which ads campaign this referral references. Use utm_id to identify a specific ads campaign.
